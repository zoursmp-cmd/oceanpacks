Frequently Asked Questions

How do I link my Minecraft account?

▼

When you visit the site, a unique 6-digit link code is generated automatically. Log in to the Minecraft server play.oceansmp.online and run the command /storeauth <code>. The website will automatically detect your login and link your character.

What currency does this store use?

▼

We use virtual server coins ($) earned purely through in-game gameplay, trading, and voting. Real-world money is strictly not accepted, and IRL trading is forbidden.

Can I gift purchases to other players?

▼

Yes! In the shopping cart sidebar, select the "Gift to another player" option, enter their exact username, and click checkout. Note that the recipient must be online on the Minecraft server to receive their items.

What happens if a transaction fails?

▼

Our store features a built-in atomic rollback system. If the items fail to deliver to your inventory for any reason (e.g. server delay), the virtual money is automatically refunded to your in-game balance instantly.

How do I earn server coins?

▼

You can earn server coins by selling resources in the global market via /shop, participating in PvE arena challenges, voting daily for the server to get bonuses, and defeating world bosses.

Are there any rank-specific discounts?

▼

No. All player ranks have access to the same prices to preserve a completely fair, competitive, and non-pay-to-win game environment for all players on the server.

How often does the store inventory rotate?

▼

The shop catalog items are updated monthly, introducing new custom key types, tools, and custom visual effects. Seasonal event items may appear during major server expansions.

Can I request a refund if I gift the wrong player?

▼

All virtual transactions are final and automatically executed on the live server. Please verify the exact spelling of the recipient's Minecraft name before checking out.

System & Testing Report

Ocean SMP Server Migration & Optimization Audit

1. Storage Recovery & World Deletion (100% Completed)

The bloated world OverworldII (~41.5 GB) was completely deleted from the remote SFTP server: region files (/region), entities database (/entities), and point-of-interest databases (/poi). A parallel SSH worker script executed 16 persistent connections simultaneously to delete over 32,000 region chunks in less than 2 minutes. All player inventories, stats, ender chests, advancements, and level files were fully preserved.

2. Red/Crimson Apple Glassmorphism Design

Rebuilt the storefront user interface to feature a modern WWDC glassmorphism styling: a glowing red-to-orange radial gradient background mesh, semi-transparent frosted card boundaries (30px backdrop filters), red highlights, micro-interactions (smooth image rotations/scaling on hover), and dynamic floating ember particles running on an interactive canvas.

3. Live Statistics & Leaderboards

Configured real-time player leaderboards using server-side scoreboards. Players can check tabs for Wealth, PvP Kills, and Playtime statistics, which pull live data from the Minecraft API alongside 3D player skin renders from mc-heads.net.

4. Secure 2FA Minecraft Authentication

To eliminate account hijackings, we removed direct username-based shop checkouts. Players now receive a unique 6-digit link code on the website and must execute /storeauth <code> inside the game. The backend polls the server-side scoreboards and immediately logs the player in once confirmed.

5. Automated Testing Suite & Validation

We created and executed a browser integration testing suite. The tests automated linking, verified item addition to shopping carts (Void Totem, Haste Potion, Strength Potion), executed the transaction checkout, verified balance subtraction (updating Vault databases), and loaded the receipt modal successfully.
